# Sanuja Salitha Andrahennadi — Portfolio

Data science student portfolio built with React, Vite, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

- **Home:** `/` — hero, projects, about, skills, and more
- **Projects:** `/projects` — project grid with filters
- **About:** `/about` — bio, metrics, education & experience
- **Skills:** `/skills` — skills & expertise (data science tools)
- **Contact:** `/contact` — dedicated contact page with form

## Customize your content

Edit **`src/data/portfolio.ts`** to update:

- Name, university, bio, email, social links
- Stats, projects, education, experience, skills
- CV download URL

Replace the profile placeholder in **`src/components/Hero.tsx`** with your photo when ready (add image to `public/` or `src/assets/`).

## Build for production

```bash
npm run build
npm run preview
```
